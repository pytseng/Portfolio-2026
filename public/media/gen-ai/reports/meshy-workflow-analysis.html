A — Navigation Bar

Logo, primary nav links (Workspace, Community, My Assets, API, Learn, Resources), credit balance, upgrade button, notifications & profile avatar.

B — Tool Sidebar

5 vertical icons for switching between major tools: Image to 3D, Model (Text to 3D), Remesh, Texture, and Animate. Vote Model at bottom.

C — Config Panel

Context-sensitive panel that changes per tool. For Text to 3D: workflow tabs, prompt input, model type, AI model, pose, generations, license, and generate button.

D — 3D Viewport

Central canvas area. Shows empty state before generation. After generation displays an interactive 3D viewer with orbit controls and model inspection tools.

E — Assets Gallery

Shows past generations grouped by project/prompt name. Includes search, upload, filter tabs by asset type, and pagination controls.

06
Post-Processing & Export
Export

From the result, you can apply additional AI workflows directly in Meshy: Retexture to change materials, Remesh to adjust topology, or Animate to add skeleton/rigging. Finally download in your desired format for use in Unreal Engine, Unity, Blender, or other 3D tools.

🎨 Retexture
Change surface materials and colors using a new text prompt. Preserves geometry.
📐 Remesh
Rebuild topology for cleaner geometry. Useful for animation-ready characters or environments.
🦴 Animate
Add automatic rigging and skeletal animation to characters (requires A-Pose or T-Pose).
↓ Export (GLB/OBJ/FBX)
Download model with PBR textures in industry-standard formats for immediate use in game engines.

UI Findings & Evaluation

UI Element
Observation
Rating
Workflow Tabs
Three icon-only tabs with no visible labels — relies on prior knowledge or tooltips. Text to 3D tab uses a "T" + cube icon that is reasonably intuitive once discovered.
Prompt Textarea
Well-designed with a 800-char counter, multilingual placeholder, and three utility actions (Copy, Reset, Save). Intuitive and spacious.
Model Type Toggle
Clear toggle between Standard and Low Poly (Beta). The "Beta" label sets appropriate expectations. Default is Standard which is sensible.
AI Model Selector
Dropdown defaults to Meshy 6 (latest). Users can select older versions but interface doesn't explain version differences in context.
Pose Selection
Premium gate clearly marked with ⭐ star icon. Three options: None, A-Pose, T-Pose. Important for character workflow but gated. Helpful for understanding animation prerequisites.
Credit Cost Display
Credit cost (20 coins) and estimated time (1 min) shown just above the generate button. Transparent and clear. Users always know the cost before committing.
Generate Button
Prominent gradient button spanning full width. Clear CTA with AI sparkle icon. Highly visible and satisfying to interact with.
Gallery / Output Panel
Right panel shows previous generations with example assets. The search bar and filter icons allow managing generated assets. Gallery persists across sessions.
03 —

Key Insights for 3D Environment Creation

💡 Insight

The Text to 3D workflow is best suited for individual objects or props rather than full sprawling environments. For 3D environments, generate modular pieces (trees, rocks, structures) separately and assemble in a 3D engine.

⚠️ Limitation

The free tier only includes 100 credits total. Each Text to 3D generation costs 20 credits, meaning only 5 generations on free tier. Plan prompts carefully and iterate thoughtfully.

✦ Prompting Tip

For environment assets, include material descriptors, scale context, and style references in your prompt. Example: "Weathered medieval stone wall segment, mossy texture, 2m tall, game-ready, low fantasy" yields better geometry.

ℹ️ Workflow Note

After generation, use the Retexture feature (also accessible from the left panel) to change materials without regenerating geometry — a powerful way to create variations of environment assets efficiently.
